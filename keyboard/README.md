# WaveKeyboard

Проект по превращению панели управления от микроволновки в **USB клавиатуру**.

При нажатии кнопок панель отправляет на компьютер стандартные события клавиатуры (HID Keyboard).

Дополнительно устройство принимает текст по отдельному HID-интерфейсу и показывает его на 4‑разрядном индикаторе.

## Соответствие кнопок на панели с тем, что отправляет в ОС

| Кнопка на панели | Что отправляется в ОС |
|---|---|
| `COOK` | `C` |
| `DEFROST` | `D` |
| `REHEAT` | `R` |
| `WAVES` | `W` |
| `TIME` | `T` |
| `ELEMENTS` | `E` |
| `PLUS` | Numpad `+` |
| `MINUS` | Numpad `-` |
| `TEN_MIN` | `Z` |
| `ONE_MIN` | `Y` |
| `TEN_SEC` | `X` |
| `STOP` | `Esc` |
| `START` | `Enter` |

## Отправка текста на дисплей (кастомный HID)

Устройство принимает входящий HID-репорт длиной **48 байт** и показывает первые 4 символа на индикаторе.

- **VID**: `0x239A`
- **PID**: `0x80B4`
- **Usage Page**: `0xFF00`
- **Usage**: `0x01`
- **Report ID**: `0x04`

Текст должен заканчиваться нулевым байтом (C строка).

Поддерживаются латинские буквы, цифры от 1 до 9, а так же символы `_`, `-` и пробел. Неподдерживаемые символы заменяются пробелом.
